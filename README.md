# 🚀 Mon Portfolio Professionnel

Une vitrine interactive présentant mon parcours, mes compétences en développement web et mes projets récents. Inspiré par un design minimaliste et moderne.

[Consulter le site en direct](https:abdelmedjidmhali.github.io/Portfolio)

---

### 🛠️ Stack Technique
* **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+)
* **Design:** Minimalisme, Responsive Design, Dark Mode
* **Outils:** Git, GitHub Pages

### 📂 Structure du Projet
- `/assets/css` : Feuilles de style personnalisées.
- `/assets/js` : Logique d'interaction et animations.
- `/assets/img` : Ressources visuelles et icônes.

### 🌟 Points Clés
- **Performance :** Optimisation du chargement des ressources.
- **Accessibilité :** Respect des normes WCAG.
- **Mobile First :** Expérience fluide sur tous les écrans.
